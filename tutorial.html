TUTORIAL

1. Pemain menekan tombol "START" untuk memulai permainan.

2. Pemain menebak apakah pernyataan tersebut benar atau salah.

3. Pilih "True" untuk pernyataan benar.

4. Pilih "False" untuk pernyataan salah.

5. Setelah selesai menjawab pilih "Submit" untuk lihat hasilnya.

6. Selamat bermain!
